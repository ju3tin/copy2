## Setup
1. Download Metamask Extension from chrome
2. Set your wallet to EVM test network
3. Get XRP from Ripple Faucet

## How to Run System
```
npm run start
```

## How to Run Tests
```
npm run test
```

## Whenever there is a change in Solidity code, use these commands
```
cd ethereum
node compile.js
node deploy.js
```
